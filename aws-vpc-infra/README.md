### About The Project
This project provide to create the VPC (Virtual Private Cloud) on AWS using Terraform.

### Used AWS Services and Features
* EC2
    - Instances
    - Security Group
    - Volumes
* VPC
    - Subnet
    - Route Tables
    - Internet Gateways
* EFS
    - AZ Specific EFS Mount
